## 安装

首先要安装Node.js，安装教程请移步[node.js安装及环境配置超详细教程【Windows系统安装包方式】\_node.js安装包-CSDN博客](https://blog.csdn.net/xuezhe5212/article/details/138576464)

安装完成后进入项目根目录终端输入命令

```
npm install
```

## 运行

```
npm run serve
```

打开 http://localhost:8080/
